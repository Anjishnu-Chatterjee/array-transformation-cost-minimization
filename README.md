# Cyclic Substring Maximum Sum

Java solution for Problem 1: Cyclic Substring Maximum Sum.

## Problem

Given a lowercase string `S`, each character has a value equal to its alphabet position:

- `a = 1`
- `b = 2`
- ...
- `z = 26`

Find the maximum possible sum of character values from any cyclic substring such that no character appears more than once.

## Example

```text
Input:
abca

Output:
6
```

## Run

```powershell
javac Main.java
'abca' | java Main
```
